# Beat the House 0.2.0 Release Checklist

> **HISTORICAL — superseded, kept for reference.** This records the shipped
> 0.2.0 release path and is not current release guidance.

This document is the source-controlled readiness ledger for the 0.2.0 demo
release candidate. Generated reports remain under `.tmp/` or Godot `user://`
paths and are not canonical release docs.

## Release Identity

- Version: `0.2.0`
- Validation run: `2026-07-01 00:20:36 -05:00`
- Itch export run: `2026-07-01 00:25 -05:00`
- Godot project version: `project.godot` application `config/version`
- Export preset versions: Windows `application/file_version` and
  `application/product_version`, Android `version/name`, iOS
  `application/short_version` and `application/version`
- Main scene: `res://scenes/main.tscn`
- Release framing: start screen states simulated gambling only, with no
  real-money wagering or cash prizes

## Included Scope

- Seeded run flow, autosave/load, daily run, content-group run packs, and
  release-safe main menu flow
- Seven full-simulation games: Pull Tabs, Slot, Bar Dice, Blackjack, Baccarat,
  Roulette, and Video Poker
- Slot cabinet art, generated slot skins, active environment slot previews, and
  slot transition performance cleanup
- Item, service, lender, event, travel, heat, debt, alcohol/luck, and terminal
  run systems
- Grand Casino demo objective with clean cashout and Pit Boss Showdown routes
- Export presets for Web, Windows, Android, and iOS
