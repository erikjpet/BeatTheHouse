# Beat the House 0.4.0 Release Checklist

Date: 2026-07-08
Owner: v0.4 Act 1 completion release closure
Version: 0.4.0

0.4.0 is the Act 1 completion release for the current source tree. It ships
the run loop, world map, performance hardening, walkable meta home, housing
progression, collection bags/loadouts, profile persistence, dialogue/talk
content, jazz/beach route content, and release packaging. The new boss
fight/final scene is explicitly outside this cut; the existing Grand Casino
clean cashout and Rourke showdown remain the Act 1 ending for 0.4.0.

## Included Scope

- Seven full game simulations: Pull Tabs, Slots, Bar Dice, Blackjack,
  Baccarat, Roulette, and Video Poker.
- Seeded runs, world-map travel, tier-2 venues, heat/debt pressure, services,
  lenders, events, challenge modifiers, and terminal Grand Casino win/loss
  paths.
- Skill-cheat actions and suspicion/heat handling across the game stack.
- Slot/pinball release work: pinball rework, Buffalo feature, real reel-shift
  nudge resolution, feature-stuck watchdog coverage, acceptance gates, and RTP
  audit coverage.
- Walkable meta home with back alley, motel room, apartment, house, and pawn
  shop rooms using the in-world room interaction path.
- Local collection bags, storage, run loadouts, run-end drops, homeless carry
  behavior, housed trade-up access, and pawn-shop selling.
- Profile persistence for schema-versioned inventory, run history, lifetime
  stats, daily/challenge state, challenge completions, and atomic writes.
- Dialogue/talk content, jazz-club route content, beach route content, semantic
  environment layouts, and attribute glyph information panels.
- Low-end/web performance fixes and fresh release-gate evidence.

## Explicit Cut

- The new boss fight/final scene is not shipped in 0.4.0.
- Android and iOS store packages are not release-ready without real signing and
  team credentials.
- Itch/GitHub publication remains a user action after the dry-run commands
  below.
