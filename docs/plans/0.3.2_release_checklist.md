# Beat the House 0.3.2 Release Checklist

Date: 2026-07-05
Owner: LD.2 release closure
Version: 0.3.2

0.3.2 is the low-end and web cleanup release for the v0.3 line. It adds no
content or player-facing features. The release work is performance hardening,
memory/load cleanup, release-gate coverage, and package hygiene.

## Release Decision

0.3.2 is ready for Web and Windows release packaging. The final tree passed the
full Godot gate, every FoundationSuite, perf/soak/determinism/stuck-state
probes, strict 60-run mouse batch, all seed audits, LD.1 web perf smoke, and
T73-TUNED12 endgame metrics reproduction. The release is version-stamped and
artifacts are packaged with hashes recorded above.
