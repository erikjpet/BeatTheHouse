# Beat the House 0.4.0 Devlog Post

## Title

Beat the House 0.4.0 - Act 1 Completion Release

## Body

0.4.0 is the Act 1 completion cut for Beat the House.

The run side is still the hard little casino climb: seven full game
simulations, seeded world-map travel, heat, debt, risky items, skill-cheat
actions, tier-2 venues, and the Grand Casino clean cashout or Rourke showdown.

The new layer is what happens around those runs. You now have a walkable meta
home, housing progression, storage, local collection bags, run loadouts,
run-end drops, profile persistence, dialogue/talk content, richer route
locations, and a pawn counter for selling spare finds.

This release also comes with fresh gate evidence: full Godot suite, every
FoundationSuite, performance, soak, determinism, stuck-state sweep, strict
mouse-batch play, web smoke, and endgame metrics all pass.

One deliberate cut: the future boss fight/final scene is not in 0.4.0. The
current Grand Casino endings remain the Act 1 finish line.

Beat the House is simulated gambling only: no real-money wagering, no cash
prizes, and no gambling monetization.

## Feature List

- Walkable home base with housing progression, storage, loadouts, and pawn-shop
  selling.
- Collection bags, run-end drops, and profile persistence across sessions.
- Dialogue and talk content for route characters and pressure moments.
- Day/night venue hours, richer route locations, and priced world-map travel.
- Stability and performance fixes for low-end play, Web audio, table surfaces,
  idle animation liveness, map selection, bankroll reveal timing, and meta home
  startup.
