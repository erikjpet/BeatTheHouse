# Beat the House 0.4 Act 1 Completion Plan

Date: 2026-07-07
Owner: v0.4 planning pass
Target: 0.4.0

0.4 is the Act 1 completion release for the current source tree, with one
explicit cut: the new boss fight/final scene is not part of this release. The
existing Grand Casino objective, clean cashout, and Rourke showdown mechanics
remain available as the current Act 1 run ending, but 0.4 should not expand
them into a new finale sequence. It should instead remove release-path shrug
copy and leave a deliberate seam for later work.

## Out Of Scope For 0.4

- New boss fight or final scene content beyond the existing Grand Casino
  cashout/showdown mechanics.
- Act 2 rooms, games, economy, or story content.
- Real prestige meta-progression; `data/prestige/purchases.json` stays empty
  unless a later owner explicitly changes the Act 1 contract.
- Steam Inventory/community market integration. Collection schema remains
  future-compatible, but the release stays local/earned-only.
- Android/iOS store submission. Presets may remain configured, but signing and
  store credentials are external.
- New casino games.

## Execution Order

1. Clean and baseline the current worktree.
2. Finish/archive web audio bridge certification.
3. Archive the completed content style guide and copy audit evidence.
4. Build the meta home environment: housing progression (alley → motel →
   apartment → house, gold-purchased, no rent yet), meta map travel, the
   walkable home replacing the collection browser (packing, opening,
   trade-up at apartment+), the pawn-shop sell counter, loadout injection
   and failure decay for normal runs only.
6. Finish profile persistence.
7. Finish the Act 1 to future-act seam.
8. Run final balance, full release gates, web/Windows smoke, and long mouse
   batch.
9. Version/package/document 0.4.0.

0.4 can ship when every queue item above is archived to `docs/todone/`, the
full gate and final balance gate pass with no unexplained failures, the release
checklist has fresh evidence, and the working tree is clean except for
deliberate build artifacts ignored by Git.
